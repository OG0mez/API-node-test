# Node Test API
 
 Node Test API creates a new Express API based on (https://github.com/OG0mez/nodejs-test) guidelines


## Installation

create a .env file 
and write inside 

``` 
PORT = 3000 
```

```npm install```

## Usage

``` npm start```
